# 迭代一
## 运行说明
1.要求已安装JDK1.8

2.BUFF文件夹下是迭代一的项目，Document文件夹下是迭代一的文档，Data文件夹下是迭代一的股票数据，executable jars文件夹下是迭代一的可执行jar文件

3.BuffServer-1.0-SNAPSHOT-jar-with-dependencies.jar和上级目录的Data文件夹是服务器端的内容，BuffClient-1.0-SNAPSHOT-jar-with-dependencies.jar和libs、text文件夹时客户端的内容

4.Windows下运行

    开始.bat

MaxOS和Linux下运行

    开始.sh

即可开始程序
